# Neovim

## Setup

```bash
git clone https://github.com/rurumimic/nvim ~/.config/nvim
```

### Alias vi

```bash
alias vi='nvim'
```

### Run Neovim

```bash
vi +MasonInstallAll
```

### Prerequisites

#### Install Packages

- neovim
- node.js
- nerd font
- ripgrep
- gcc
- make

#### Clean old neovim folders

```bash
rm -rf ~/.config/nvim
rm -rf ~/.local/state/nvim
rm -rf ~/.local/share/nvim
```
