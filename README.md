## Update NvChad

```bash
:NvChadUpdate
```

## My Custom Configurations

```bash
rm -rf ~/.config/nvim/lua/custom
git clone https://github.com/rurumimic/nvim ~/.config/nvim/lua/custom --depth 1
git fetch && git pull
```
